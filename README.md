To re-generate the list above:

```bash
find . ! \( -type d -name .git -prune \) \
    -type f ! -name README.md -print0 |
    xargs -0 grep -Pho '((?<=\$\{)|(?<=lk_is_true )|(?<=lk_is_false ))LK_[a-zA-Z0-9_]+\b(?!\})' |
    sort -u |
    sed 's/.*/- `&`/'
```
